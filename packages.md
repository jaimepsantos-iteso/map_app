### Required packages

- PyQt5
- PyQtWebEngine
- networkx
- osmnx
- pandas
- geopandas
- folium
- branca
- matplotlib
- mapclassify

Suggested installs:

- Conda (recommended, uses conda-forge):
    - conda install -c conda-forge pyqt pyqtwebengine networkx osmnx pyosmium pandas geopandas folium 

- Pip (virtualenv):
    - pip install PyQt5 PyQtWebEngine networkx osmnx pyosmium pandas geopandas folium

Note: use conda-forge for geospatial packages to avoid dependency issues.
